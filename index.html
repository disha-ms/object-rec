Communication Flow

How the alert system operates

  1. Trigger: Panic button is pressed
  2. Primary Alert: System sends SMS to all configured contacts with location details
  3. Backup Check: If GSM network is unavailable, system switches to LoRa transmission
  4. Confirmation: System provides visual feedback that alert was sent
  5. App Sync: Mobile app synchronizes contacts via Bluetooth when in range
